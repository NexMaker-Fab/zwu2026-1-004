HOMEWORK 1

Project Management - Prepare

Homework 1: Prepare

1. 准备工作

我们将使用以下工具来构建我们的网页:

  • Git: 用于版本控制。Mac系统无需安装,主要用于配置。
  • GitHub: 用作我们网页的托管服务。
  • GitHub Desktop: 使我们能够轻松地将代码从本地上传到GitHub。
  • 通义灵码: 用于编写文档和代码。

2. 网页设置

注意:仓库是原先已经建好的,下面是操作示范。

在 GitHub 上创建新仓库以托管您的网站

创建一个公开仓库,您可以与团队成员协作创建网页内容。

在 GitHub 上创建新的公开仓库

配置 GitHub Pages

将页面设置为空。点击仓库的设置,在左侧选择"Pages"。选择"main"分支和"/root"目录并保存。上方的链接允许您查看页面。

Configure GitHub Pages - Access Pages settings from repository Settings
Select main branch and /root directory, then save

添加协作者

点击"Settings",在左侧找到"Collaborators",然后点击"Add people",输入他们的邮箱地址,并设置他们的角色(maintainer)。

点击 Settings → Collaborators → Add people
输入邮箱地址并设置 Maintainer 角色

3. 克隆仓库到本地

打开 GitHub Desktop 并登录

打开软件并登录您的 GitHub 账户。

克隆仓库

克隆仓库并选择一个位置保存到您的计算机。

Clone the repository to your local computer

7. 项目文件与目录说明

根目录文件

  • index.html - 网站主页，包含视频背景、团队介绍、作业列表和最终项目展示
  • style.css - 全局样式文件，定义网站的视觉风格、颜色、字体和布局
  • upload.js - 上传功能JavaScript脚本，处理文件上传逻辑
  • about.html - 关于页面，介绍项目和团队信息
  • team.html - 团队介绍页面，展示所有成员信息
  • homework.html - 作业列表页面，汇总所有平时作业链接
  • homework1.html ~ homework9.html - 九个平时作业的详细内容页面
  • profile-chen.html, chen.html, wang.html, li.html, zhang.html, zhu.html, he.html, xu.html - 团队成员个人介绍页面
  • README.md - 项目说明文档，包含项目介绍、技术栈和使用说明
  • .gitignore - Git忽略文件配置，指定不需要版本控制的文件

图片资源目录 (image/)

  • logo.png - 网站Logo图标
  • homework1/ - 作业一的截图和图片资源（创建仓库、Pages设置、添加协作者等）
  • homework2/ - 作业二的截图和图片资源（Arduino IDE、电路连接、核心组件等）
  • homework4/ - 作业四的截图和图片资源（Fusion 360建模、装配、磁吸底座设计等）

视频资源目录 (videos/)

  • homework2/flow-lamp-demo.mp4 - 作业二的流水灯演示视频
  • homework4/ - 作业四的相关视频资源（如有）

GitHub工作流目录 (.github/workflows/)

  • pages-rebuild.txt - GitHub Pages自动部署配置文件，用于触发网站重新构建

其他辅助文件

  • deploy.bat - Windows批处理脚本，用于快速部署网站
  • check-deployment.ps1, fix-deployment.ps1, optimize-favicon.ps1 - PowerShell脚本，用于检查、修复部署和优化favicon
  • create-independent-site.ps1 - 创建独立站点的PowerShell脚本
  • DEPLOYMENT_CHECKLIST.md, DEPLOYMENT_COMPLETE.md, DEPLOY_FIX_GUIDE.md - 部署相关文档
  • FEATURE_UPGRADE_COMPLETE.md, MENU_OPTIMIZATION_COMPLETE.md - 功能升级和优化完成记录
  • HOMEWORK1_IMAGE_GUIDE.md, INDEPENDENT_SITE_GUIDE.md, UPLOAD_FEATURE_GUIDE.md - 特定功能使用指南
  • PROFILE_LINK_DEBUG.md, FIX_DEPLOYMENT.md - 问题修复和调试文档
