Homework 1: Prepare

1. Prepare

We will use the following tools to build our own webpage:

  • Git: Used for version control. No installation required on Mac, mainly for configuration.
  • GitHub: Used as the hosting service for our webpage.
  • GitHub Desktop: Makes it easy for us to upload code from local to GitHub.
  • Lingma: Used for writing documents and code.

Configure GitHub Pages

Set the page to be empty. Click on the settings of the repository and select "Pages" on the left. Choose "main" and "/root" and save. The link above allows you to view the page.

内容概览 - Content Overview

本文档包含以下内容：

  • 1. Tools and Environment Setup（工具与环境设置）：介绍本次作业使用的工具和软件环境，包括Git、GitHub、GitHub Desktop和Lingma AI助手。
  • 2. Web page setting（网页设置）：演示如何在GitHub上创建仓库、配置GitHub Pages以及添加协作者。注意：仓库是原先已经建好的，这里是操作示范。
  • 3. Clone the repository to GitHub（克隆仓库到本地）：展示如何使用GitHub Desktop将远程仓库克隆到本地计算机进行开发。
  • 4. Use Lingma for Website Development（使用Lingma进行网站开发）：介绍如何使用Lingma AI助手辅助网站开发，包含完整的提示词示例，展示如何通过AI生成符合圣罗兰风格的品牌官网代码。
  • 5. Modify Files and Commit（修改文件并提交）：演示如何编写提交说明并记录代码更改。
  • 6. Push to GitHub（推送到GitHub）：展示如何将本地更改推送到GitHub远程仓库。
  • 7. Project Directory Structure Organization（项目目录结构整理）：说明如何组织项目目录结构，创建专门的文件夹存放文档、图片、视频等资源，提高项目的可维护性。

学习目标：通过本次作业，您将掌握使用Git进行版本控制、使用GitHub托管项目、使用GitHub Desktop管理代码、以及利用AI助手提高开发效率的完整工作流程。
