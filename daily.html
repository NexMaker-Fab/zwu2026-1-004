提交流程

1. 查看要求

点击作业卡片查看详细的任务说明、技术要求和评估标准。

2. 完成作业

按照要求完成设计和制作，记录整个过程并拍摄照片或视频。

3. 准备文档

编写详细的项目文档，包括设计思路、制作步骤、遇到的问题和解决方案。

4. 提交审核

通过 GitHub 提交所有文件，并在系统中更新作业状态为"已完成"。
