## 💡 为什么不能移动？

GitHub Pages 直接将根目录的文件映射到网站 URL。如果移动文件：
1. 需要修改所有引用这些文件的链接
2. 会破坏现有的导航结构
3. 可能导致页面无法访问

因此，保持所有 HTML 文件在根目录是最安全和最简单的方案。
