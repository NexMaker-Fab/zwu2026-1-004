1. Learn Open Source Hardware

Arduino is an open-source electronics prototyping platform that consists of hardware (development boards), software (IDE), and a community, enabling absolute beginners to quickly build interactive electronics, robotics, and IoT projects.

Arduino 核心组件概览

Core Components

Hardware (Development Boards)

  • Mainstream models: Arduino Uno (most commonly used), Nano, Mega, ESP32 (with WiFi).
  • Onboard features: USB port, power supply, microcontroller, digital/analog I/O pins; supports connection to sensors, motors, and LEDs.

Open Source

What is Open Source?

Open source means the source code of software/hardware is publicly available. It represents a philosophy and methodology where resources are shared openly for anyone to access, study, modify, and distribute.

Key Principles of Open Source:
  • Freedom to Use: Anyone can use the software/hardware for any purpose without restrictions.
  • Freedom to Study: The source code/schematics are available for learning and understanding how it works.
  • Freedom to Modify: Users can adapt and customize the design to meet their specific needs.
  • Freedom to Share: Modified versions can be redistributed, benefiting the entire community.
Arduino's Open Source Ecosystem:
  • Hardware Schematics: All Arduino board designs are published under Creative Commons licenses, allowing anyone to manufacture compatible boards.
  • Software & Libraries: The Arduino IDE and core libraries are open-source (LGPL/GPL licenses), enabling developers to extend functionality.
  • Community Contributions: Code, projects, and tutorials are shared by millions of users worldwide through platforms like GitHub, Arduino Project Hub, and forums.
  • Collaborative Development: Bugs are fixed faster, features are added continuously, and innovations emerge from community collaboration.
Benefits of Open Source:
  • Transparency: Full visibility into how things work builds trust and enables learning.
  • Cost-Effective: No licensing fees; reduces development costs significantly.
  • Rapid Innovation: Community-driven improvements accelerate technological advancement.
  • Educational Value: Perfect for teaching electronics, programming, and engineering concepts.
  • Sustainability: Designs remain accessible even if original manufacturers discontinue products.

Key Features

  • Easy to learn: no complex register knowledge required; an LED can be lit with just a few lines of code.
  • Low cost: compatible boards cost dozens of yuan, with inexpensive modules.
  • Highly extensible: supports a wide range of sensors, motors, and expansion shields.
  • Cross-platform: the IDE works on three major operating systems.
  • Strong community: solutions can be found for almost any problem.

Typical Applications

  • Basic: LED blinking, push buttons, light sensor control, buzzers
  • Intermediate: Bluetooth / WiFi remote-controlled cars, environmental monitoring (temperature/humidity / PM2.5), smart home devices
  • Internet of Things (IoT): ESP32 connecting to cloud platforms, control via mobile apps

Arduino UNO Q Introduction

Arduino UNO Q is a next-generation development board that combines the simplicity of traditional Arduino with powerful computing capabilities.

1. Core Structure

Dual-chip design, it works as both a microcontroller and a mini computer.

2. Two Chips' Duties

  • Quad-core Qualcomm chip: runs Linux for AI, vision and network tasks
  • STM32 chip: controls sensors and motors in real time

3. Hardware Compatibility

It shares the same pin layout as standard UNO, compatible with old shields and sensors.

4. Built-in Specs

  • Comes with 2/4GB RAM
  • 16/32GB storage
  • Wi-Fi 5 and Bluetooth 5.1 onboard

5. Unique Hardware

  • Equipped with an 8×13 grayscale LED matrix
  • USB-C supports power, data transmission and video output

6. Development Support

One unified tool for Arduino sketches, Python scripts and AI model programming.

7. Suitable Projects

  • Ideal for AI vision robots
  • Voice devices
  • Smart IoT terminals

Operation Code

/*
 * 声音控制三色灯 - 取峰值，绿灯范围可调
 * 安静 (<60)     : 流水灯 (绿->红->黄)
 * 正常 (60~350)  : 绿灯常亮
 * 大声 (>350)    : 黄灯呼吸灯
 */

const int PIN_LED_GREEN = A0;
const int PIN_LED_RED   = A1;
const int PIN_LED_YELLOW = 9;
const int SOUND_SENSOR = A3;

// ========== 可调整的阈值 ==========
const int SILENT_THRESHOLD = 60;    // 小于此值 → 安静模式
const int LOUD_THRESHOLD   = 350;   // 大于此值 → 大声模式 (调高此值可让绿灯范围更大)
// =================================

const unsigned long INTERVAL = 300;      // 流水灯切换间隔(ms)
const unsigned long BREATHE_CYCLE = 2000; // 呼吸周期(ms)

const int NUM_SAMPLES = 10;        // 每次采样次数
const int REQUIRED_STABLE = 3;     // 连续稳定次数

unsigned long lastChange = 0;
int step = 0;
unsigned long breatheStart = 0;

int lastMode = -1;
int stableCount = 0;

// 读取峰值（最大值）
int readMaxVolume() {
  int maxVal = 0;
  for (int i = 0; i < NUM_SAMPLES; i++) {
    int v = analogRead(SOUND_SENSOR);
    if (v > maxVal) maxVal = v;
    delay(3);
  }
  return maxVal;
}

void setup() {
  pinMode(PIN_LED_GREEN, OUTPUT);
  pinMode(PIN_LED_RED, OUTPUT);
  pinMode(PIN_LED_YELLOW, OUTPUT);
  Serial.begin(9600);
  Serial.println("Ready - Peak detection, LOUD_THRESHOLD=350");
  digitalWrite(PIN_LED_GREEN, LOW);
  digitalWrite(PIN_LED_RED, LOW);
  analogWrite(PIN_LED_YELLOW, 0);
}

void loop() {
  int volume = readMaxVolume();
  Serial.println(volume);

  int mode;
  if (volume < SILENT_THRESHOLD)
    mode = 0;
  else if (volume < LOUD_THRESHOLD)
    mode = 1;
  else
    mode = 2;

  if (mode == lastMode) {
    stableCount++;
  } else {
    stableCount = 0;
    lastMode = mode;
  }

  if (stableCount >= REQUIRED_STABLE) {
    // 熄灭所有灯
    digitalWrite(PIN_LED_GREEN, LOW);
    digitalWrite(PIN_LED_RED, LOW);
    analogWrite(PIN_LED_YELLOW, 0);

    if (mode == 0) {
      // 安静模式：流水灯
      unsigned long now = millis();
      if (now - lastChange >= INTERVAL) {
        lastChange = now;
        step = (step + 1) % 3;
        if (step == 0) digitalWrite(PIN_LED_GREEN, HIGH);
        else if (step == 1) digitalWrite(PIN_LED_RED, HIGH);
        else analogWrite(PIN_LED_YELLOW, 255);
      }
    }
    else if (mode == 1) {
      // 正常模式：绿灯常亮
      digitalWrite(PIN_LED_GREEN, HIGH);
    }
    else {
      // 大声模式：黄灯呼吸灯
      if (breatheStart == 0) breatheStart = millis();
      unsigned long elapsed = millis() - breatheStart;
      float t = (float)(elapsed % BREATHE_CYCLE) / BREATHE_CYCLE;
      int brightness;
      if (t < 0.5) {
        brightness = (int)(t * 2 * 255);
      } else {
        brightness = (int)((1 - t) * 2 * 255);
      }
      analogWrite(PIN_LED_YELLOW, brightness);
    }
  }
  delay(30);
}
